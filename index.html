Gitbash/GitHub


Steps to create repo using Git -

  1. Install git
  2. Configure git
  4. Create a repo on github
  5. Clone the repo in the local system $git clone #repo link
  6. navigate into the cloned file using OS commands $cd $ls $cd .. $touch
  7. after accessing the file $code . to open code editor
  8. do the coding stuff, now to save this code changes to git we have to use $git add $git commit -m/-am $git push
  9. $git status to know the details of the local repo
  10. to add files and folders, we gotta them to $git add like $git add images/ (this selects the whole folder)
